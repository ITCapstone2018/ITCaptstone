Stored Procedures and Triggers Sub-module

This reference is intended to show you how to use the various topics of the "Stored Procedures and Triggers" Sub-module. Choose a topic from below to be directed to its associated help content.

Procedures, Functions, and Triggers 
Definition and Explanation

The Procedures and Functions scenarios both work similarly. Upon entering each scenario, you will be taken an Introduction page with the definition, structure, and syntax of each concept. To proceed to the exercise, click on the "Example" tab at the top of the screen.

Create the Stored Procedure / Function / Trigger

The scenario will show an example of a Stored Procedure or Function created with SQL. To create a sample stored procedure, simply press the Create a Stored Procedure" button. Likewise, the Function scenario works exactly in the same manner, but the button will read "Create a Function" instead. Regardless, after pressing the button, the scenario will provide you with SQL used to create the stored procedure or function, and the related tables will be shown on the right.

Call the Stored Procedure / Function or Activate the Trigger

Once the SQL to create the Stored Procedure has been generated, the "Call the Stored Procedure (or Function)" button can be clicked on to show how it is called within a database. Once clicked, the SQL line calling the Stored Procedure and the tables associated with it are displayed. The Triggers scenario works similarly, but the "Call the Stored Procedure" button is replaced with "Activate the Trigger".

"Triggers & Locks" and Rights 
Select a Scenario 

Using the tabbed navigation at the top, select a scenario. By default, "Trigger and Error" will be displayed.

Walk Through the Scenario

Use the "Next Step" button to walk through the steps of the scenario. While clicking "Next Step", the screen will fill up with information describing and depicting what is happening in the database.

Resetting

When the end of the scenario has been reached, the "Next Step" button will turn gray. To work through the scenario again, click the "Reset" button.
