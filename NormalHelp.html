Normalization Sub-Module

This reference is intended to show you how to use the "Normalization" Sub-module.

Read the Functional Dependencies

First, read the instructions, and then use the functional dependencies on the right to plan your tables.

Create a Table

Once you have planned your tables, use your cursor to highlight and select the elements that you want in the first table. When you have highlighted every element that you want included in that table, click on the "Create Table" button as shown with the red box.

Designating Primary Keys and Solving the Exercise

By clicking "Create Table," the elements you had selected will appear below in their own table as shown.

To select the primary key of a table, click on the column or columns that create the primary key. Once clicked, the column will highlight, distinguishing that column as the primary key of the table.

Repeat these steps until you have created all tables needed to completely normalize the data. Once finished, ensure the primary keys have been selected for each table, and then click on the Finish button. Feedback on your solution will replace the instructions box.

If you didn't completely get the solution correct, keep trying! Use the provided feedback to refine your solution. If you think it best to completely start over, a Reset button is provided near the Finish and Create Table buttons.
