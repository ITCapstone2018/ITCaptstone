ER Notations Sub-Module

The ER Notation Sub-Module is no longer interactive.

Basic Operation

This submodule will create E-R Diagrams based on specified Relationship and Notation types. To start generating E-R Diagrams, first choose a Relationship type from the left, and then a Notation set from the right as depicted below with the red boxes. Once you have chosen both the Relationship type and the Notation set, the E-R Diagram will be generated below.

Choosing Multiple Notations

It is possible to chose more than one Notation to perform a side by side comparison of multiple sets. To choose more than one Notation type, hold down the shift key while selecting types from the box on the right. Multiple ER diagrams will be drawn below.
