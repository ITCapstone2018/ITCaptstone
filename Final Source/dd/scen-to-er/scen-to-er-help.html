Help

Scenario to ER Sub-module

This reference is intended to show you how to use the "Scenario to ER" Sub-module.

Read the Scenario

Read the Scenario Description or examine the Table or Form displayed in the upper portion of the screen.

OR

Examine Each Solution

Click the "Solution" buttons underneath the Scenario Description to view each solution. Click the "Show All" button on the right side of the screen to view all four solutions at the same time.

Choose a Solution

After you've selected the solution you want to choose, click the "Choose Solution" button at the bottom center of the screen. A feedback box will appear telling you if you are correct, and offering some help if you've chosen the wrong solution.

Get Help

To see a hint, click the "Hint" button on the bottom left of the screen.

Try All the Scenarios

Use the navigation panel on the left of the screen to choose the Scenario for which you are choosing the appropriate Entity-Relationship diagram. There are several scenarios, so be sure to try them all!
