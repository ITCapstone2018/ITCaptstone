Help

SQL Pseudocode Sub-module

This reference is intended to show you how to use the various topics of the "SQL Pseudocode" Sub-module. Choose a topic from below to be directed to its associated help content.

Basic and Advanced Code
Select a Query Type  

Start by choosing a query type from the Scenario's tabbed navigation bar at top with each tab representing a different type. By default, the first query type of each scenario will be opened, or Projection for Code - Basic and Multiplication for Code - Advanced.

Observe the Pseudocode Process

To move through the pseudocode process for each given SQL statement, click the "Next Step" button. Alternatively, you can use the "Timer Start" button to automatically move through each step of the demonstration. For each step, the Code window will display the pseudocode that a database cycles through to process an SQL statement, and the table generated by the database will be displayed in the results table.

Resetting

You can use the "Reset" button at anytime during the demonstration to start again from the beginning. When the end of the scenario has been reached, the "Next Step" button will disappear. To work through the scenario again, click the "Reset" button.

Join and Minus Code
Observe the Pseudocode Process

To move through the pseudocode process for each given set of SQL statements, click the "Next Step" button. The Relational Algebra window will display the pseudocode that a database cycles through to process the SQL. At the bottom of the screen, temporary tables will graphically illustrate how the database is working through the relational algebra pseudocode.

Manage Windows

This scenario allows windows to be easily managed from the menu bar. By clicking the on the Window menu bar, at the top of the scenario, windows can be easily toggled on or off to manage screen space.

Resetting

When the end of the scenario has been reached, the "Next Step" button will disappear. To work through the scenario again, click the "Reset" button.
